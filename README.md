# retailflow-data-pipeline
RetailFlow Data Pipeline Graduation Project | DEPI - Data Engineering Track 
Project Status Planning Phase.
This repository is being prepared for the DEPI project. Our team is building a robust, automated data pipeline for analytics and dashboarding. 
Project Objective The goal of RetailFlow is to build an end-to-end ETL pipeline that: Extracts retail data (Sales, Customers, Products). 
Transforms and cleans data to ensure quality. Loads data into a PostgreSQL warehouse optimized for queries.
Visualizes key performance indicators through a Dashboard. 
Our Team A: Data Ingestion - Architecture & Loading 
         B: ETL Scripts - Visualization, Cleaning & Generation 
         C: ETL Extract - Architecture, Sources & Logic 
         D: Database - Optimization & Docs 
         E: Dashboard - Visualization 
         F: Documentation - Schema Design & Generation
